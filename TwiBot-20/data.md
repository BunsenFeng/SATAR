## TwiBot-20

After obtaining permission, you can visit link https://drive.google.com/drive/folders/1ju-bXk6X3n7FNNzv1aXH7-zgqAAnid1N?usp=sharing to download the complete data set.

File named "tweet_sample.csv" cintains a sample of semantics information and each row contains the ID of a user and its last 200 tweets. File named "profile_sample.csv" cintains a sample of property information and each row contains the ID of a user and its all property information collected by Twitter API. File named "neighbor_sample.csv" cintains a sample of neighborhood information and each row contains the ID of a user and its followings' IDs or followers' IDs.
